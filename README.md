# 超星AI学习助手

超星学习平台答题辅助脚本，自动识别单选题/多选题/判断题，调用 DeepSeek AI 智能作答。

## 功能特性

- 🔍 自动识别页面中所有题目
- 🤖 调用 DeepSeek API 获取答案
- ✅ 支持单选、多选、判断题
- ❌ 不支持填空题简答题
- 📊 答题进度实时显示
- 🔎 答案复核验证
- ➡️ 一键跳转下一题

## 安装方法

1. 安装 Tampermonkey 浏览器扩展
2. 点击 [此处安装脚本](https://raw.githubusercontent.com/你的用户名/仓库名/main/脚本文件名.user.js)（将链接中的用户名和仓库名改为你自己的）
3. 打开超星课程页面，右下角出现 AI 助手面板
4. 点击"配置 API Key"，输入 DeepSeek API Key

## 获取 API Key

访问 [DeepSeek 开放平台](https://platform.deepseek.com/) 注册账号，在 API Keys 页面生成密钥。

## 使用说明

1. 进入超星作业/考试页面
2. 点击 **智能作答**，AI 自动填写答案
3. 点击 **答案复核**，AI 检查是否正确
4. 点击 **跳转下一题** 继续答题

## 注意事项

- ⚠️ AI 答案仅供参考，请人工复核确认
- 📖 建议用于学习辅助，遵守学术诚信
